Welcome to Task Manager

This application will help you stay organized!

Most features require logging in. Users who are not authorized have access to the welcome screen, as well as the log in or registration page.

Admin (Teacher) Can:

  • Create a task & assign a task to any user.
  • View a list of all users with the possibility to delete users.
  • View a list of all tasks with the possibility to edit or delete tasks.

Common User (Student) Can:

  • Create a task only for himself/herself.
  • View a list of all users with no actions allowed.
  • View a list of all tasks but edit or delete only tasks for which he/she is responsible.

Every Authorized User Can:

  • View his/her own profile.
  • Mark a task as completed/uncompleted.
